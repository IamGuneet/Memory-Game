Memory Game

How to play?

  • Press any key to start the game.
  • The computer will press a button of any color.
  • Click on the same button and the head will displpay sucess.
  • The computer will then press another button and now you have to press the buttons from the starting pattern
  • The patterns will keep on getting harder with increasing levels.
  • On wrong move press any key to start again.
  • Enjoy!
